Frontend Development

Frontend development is the process of developing the graphical user interface (GUI) of a website. It is achieved through the use of HTML, CSS, and JavaScript. The purpose of frontend development is to create a website that users can easily view and interact with.

  • HTML: The HyperText Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It structures the content of a webpage.

  • CSS: Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML or XML. CSS helps in defining how HTML elements should be displayed on screen.

    1. Pure CSS

    2. Bootstrap (Framework)

    3. Tailwind (Framework)

    4. Sass (Preprocessor)


  • JavaScript: JavaScript (JS) is a programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS. It is used to add functionality and interactivity to a website.

    1. Vanilla JS

    2. jQuery (Library)

    3. React (Framework)

    4. Angular (Framework)

    5. Vue (Framework)

    6. TypeScript (Preprocessor)
